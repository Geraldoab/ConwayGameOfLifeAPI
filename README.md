# Conway's Game Of Life API
### https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life
An implementation of Conway's Game of Life using C# and .NET 7.0

- DDD
- SOLID
- SWAGGER
- Strategy Pattern
- Unit Tests
- Local Response Cache
- Filters
